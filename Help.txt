도움말:

1번 게임 : 홀/짝 게임
2번 게임 : 업다운 게임
3번 게임 : 숫자 야구 게임
4 : 게임종료

1,2,3,4 외의 모든 숫자나 문자는 Error 발생



1번 게임 : 홀/짝 게임 설명

1-1. 게임시작 후 게임플레이어는 10000$를 받는 다.
1-2.  자신이 원하는 수의 배팅을 건 후 홀/짝 둘중 하나를 선택한다.
1-3. 배팅을 한 후 게임을 더 할것이면 아무키나 누르고 게임을 그만 할 것이면 Stop을 누른다.
승리 시 자신이 배팅한 금액의 2배를 돌려받는다. 
패배 시 자신이 배팅한 금액을 잃는다.
자신이 들고 있던 금액을 모두 잃을 시 게임은 자동으로 종료 된다.

** Stop을 입력할 시 S대문자 top 소문자로 입력하여 Stop를 입력해주세요
** Odd, Even 또한 같이 입력해주세요.



2번 게임 : 업다운 게임 설명

2-1. 1~100 사이의 숫자를 플레이어가 선택하면 컴퓨터가 지정한 숫자가 
플레이어가 선택한 숫자보다 UP에 있는지 Down에 있는지 알려줍니다.



3번게임 : 숫자 야구 게임

Ball : 자신이 선택한 3자리 수는 존재하나 자리는 맞지 않은 개수
Strike : 자신이 선택한 3자리 수가 자리와 숫자가 맞은 개수
out : 자신이 선택한 3자리 수가 모두 틀린 것

ex) 컴퓨터가 설정한 숫자 : 567
3-1. 123 입력시 1out
3-2. 456 입력시 2ball
3-3. 789 입력시 1ball
3-4. 547 입력시 2Strike
이렇게 숫자를 모두 다 맞추었을때는 3Strike로 게임이 종료가 된다.
